# F3 – Conditionals (Type 0 / 1 / 2)

> Mục tiêu task: nói/viết đúng “nếu… thì…” theo 3 mức: fact (0), real future (1), unreal now (2).

---

## Purpose

Task này giúp bạn chọn đúng loại câu điều kiện để diễn đạt: **sự thật/định luật**, **khả năng có thật trong tương lai**, và **giả định không thật ở hiện tại**. Dùng đúng type làm câu nghe tự nhiên và logic.

## Key points

- Type 0 = **fact/always true** (thói quen, quy luật, hướng dẫn): hiện tại–hiện tại.
- Type 1 = **real possibility** (tương lai có thể xảy ra): present + will.
- Type 2 = **unreal now** (giả định trái hiện tại): past simple + would.
- Type 2 dùng `were` cho **I/he/she/it** trong văn chuẩn: `If I were...` (thực tế giao tiếp có khi dùng `was`, nhưng task này ưu tiên `were`).
- `if`-clause không dùng `will` trong Type 1: ❌ `If it will rain...`.
- Dấu phẩy: nếu `If-clause` đứng đầu câu → thường có `,`.
- `unless` = `if not` (thường dùng trong Type 1/0).
- Nhận diện nhanh: Type 2 thường có tín hiệu “không thật”: *If I had more time / If I were you / If I knew...*

## Rule summary

- **Type 0 (fact):** `If + present simple, present simple.`
  - If you heat ice, it melts.

- **Type 1 (real future):** `If + present simple, will + V1.`
  - If it rains, we will stay home.

- **Type 2 (unreal now):** `If + past simple, would + V1.`
  - If I had time, I would help you.

## Examples

**Type 0 – facts / routines**
1) If you mix red and blue, you get purple.
2) If I drink coffee at night, I can’t sleep.

**Type 1 – real future / likely**
3) If you study today, you will feel better tomorrow.
4) If it rains this afternoon, we will cancel the picnic.
5) If you don’t hurry, you will miss the bus.

**Type 2 – unreal now / hypothetical**
6) If I were you, I would talk to the manager.
7) If I had more money, I would buy a new laptop.
8) If she knew the answer, she would tell you.

## Common mistakes

1) ❌ If it will rain, we will stay home.
   - ✅ If it rains, we will stay home.

2) ❌ If I will have time, I help you.
   - ✅ If I have time, I will help you.

3) ❌ If I am you, I will do it.
   - ✅ If I were you, I would do it. (Type 2)

4) ❌ If I had time, I will help you.
   - ✅ If I have time, I will help you. (Type 1)
   - ✅ If I had time, I would help you. (Type 2)

5) ❌ If you heat ice, it will melt.
   - ✅ If you heat ice, it melts. (Type 0: fact)

6) ❌ If she would study, she would pass.
   - ✅ If she studied, she would pass.

## Practice A

Chọn/điền đúng. Mỗi câu chọn **Type 0/1/2** và viết câu đúng ngữ pháp.

1) (Type __) If you ___ (not water) plants, they ___ (die).
2) (Type __) If I ___ (have) free time tonight, I ___ (call) you.
3) (Type __) If I ___ (be) taller, I ___ (play) basketball.
4) (Type __) If you ___ (touch) fire, you ___ (get) burned.
5) (Type __) If she ___ (study) harder, she ___ (pass) the exam. (giả định hiện tại)

6) Chọn câu đúng:
   A) If it will be sunny tomorrow, we will go hiking.
   B) If it is sunny tomorrow, we will go hiking.

7) Chọn câu đúng:
   A) If I were you, I would apologize.
   B) If I am you, I will apologize.

8) Viết lại dùng `unless` (giữ nghĩa, Type 1):
   If you don’t leave now, you will be late.

9) Chọn type đúng cho câu:
   If I drink too much tea, I can’t sleep.
   A) Type 0  B) Type 1  C) Type 2

10) Chọn type đúng cho câu:
    If we finish early, we will go for coffee.
    A) Type 0  B) Type 1  C) Type 2

11) Chọn type đúng cho câu:
    If he had a car, he would drive to work.
    A) Type 0  B) Type 1  C) Type 2

12) Sửa lỗi:
    If she will call me, I will answer.

13) Sửa lỗi:
    If I knew his number, I will text him.

14) Hoàn thành câu (Type 2):
    If I ___ (know) the answer, I ___ (tell) you.

15) Hoàn thành câu (Type 0):
    If you ___ (add) sugar, the tea ___ (taste) sweeter.

## Answer key

1) Type 0: If you **don’t water** plants, they **die**. — thói quen/định luật.
2) Type 1: If I **have** free time tonight, I **will call** you. — khả năng có thật.
3) Type 2: If I **were** taller, I **would play** basketball. — giả định trái hiện tại.
4) Type 0: If you **touch** fire, you **get** burned. — fact.
5) Type 2: If she **studied** harder, she **would pass** the exam. — giả định hiện tại.

6) **B** — `if`-clause Type 1 không dùng `will`.
7) **A** — Type 2: `If I were you, I would...`.

8) You will be late **unless** you leave now. — `unless` = `if not`.

9) **A (Type 0)** — thói quen/kết quả luôn đúng với điều kiện.
10) **B (Type 1)** — điều kiện tương lai có thể xảy ra.
11) **C (Type 2)** — `had/would` = unreal now.

12) ✅ If she **calls** me, I will answer.
13) ✅ If I knew his number, I **would** text him.
14) ✅ If I **knew** the answer, I **would tell** you.
15) ✅ If you **add** sugar, the tea **tastes** sweeter.

## Practice B

Viết theo template. Mỗi đề viết 3–4 câu.

1) Type 0: nêu 2 “rules”/thói quen của bạn (ví dụ học, sức khỏe).
2) Type 1: 3 kế hoạch tuần này (nếu… thì…) có khả năng thật.
3) Type 2: 3 giả định “unreal now” về cuộc sống/công việc/học tập.
4) Mix: viết 1 đoạn 6 câu có đủ Type 0, 1, 2 (mỗi loại ít nhất 1 câu).
5) Advice: dùng mẫu `If I were you, I would...` cho 3 lời khuyên.

## Self-check rubric

- Tôi phân biệt được fact (0) vs real future (1) vs unreal now (2).
- Tôi dùng đúng form: 0 (present-present), 1 (present-will), 2 (past-would).
- Tôi không dùng `will` trong mệnh đề `if` của Type 1.
- Tôi dùng `were` với I/he/she/it trong Type 2 (chuẩn).
- Tôi đặt dấu phẩy đúng khi `If-clause` đứng đầu câu.
