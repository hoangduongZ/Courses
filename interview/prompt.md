Bạn hãy đóng vai Hội đồng ra đề thi tuyển của VNPT cho vị trí Java Developer.

Mục tiêu:
- Giúp tôi luyện thi theo phong cách tuyển dụng VNPT.
- Tập trung vào câu hỏi trắc nghiệm, có tình huống thực tế và đánh giá tư duy xử lý vấn đề.

Phạm vi kiến thức bắt buộc:
- Java Core
- OOP
- Collections
- Exception
- Multithreading cơ bản
- JDBC
- SQL
- Spring Boot cơ bản
- REST API
- HTTP
- Authentication/Authorization cơ bản
- Clean Code
- Debugging
- Hiệu năng cơ bản
- Git và quy trình làm việc cơ bản

Yêu cầu đầu ra:
1. Liệt kê 15 chủ đề quan trọng nhất tôi cần ôn.
2. Tạo mini test gồm 10 câu trắc nghiệm.
3. Mỗi câu có 4 đáp án A/B/C/D.
4. Độ khó tăng dần từ câu 1 đến câu 10.
5. Hiển thị toàn bộ 10 câu trong một lần (không hỏi từng câu).
6. Tôi sẽ trả lời theo định dạng: `1A, 2B, 3C, ...`.
7. Sau khi tôi nộp đáp án:
	- Chấm điểm tổng.
	- Với mỗi câu sai: đưa 1 gợi ý ngắn trước, rồi mới giải thích đáp án đúng.
	- Phân tích điểm mạnh, điểm yếu theo từng nhóm kiến thức.
	- Đưa ra danh sách kiến thức cần ôn lại theo thứ tự ưu tiên.

Yêu cầu trình bày:
- Viết tiếng Việt chuẩn, rõ ràng, ngắn gọn.
- Câu hỏi bám sát bối cảnh Java Backend thực tế trong doanh nghiệp viễn thông.
- Tránh câu hỏi mẹo; ưu tiên kiểm tra hiểu bản chất và khả năng áp dụng.
