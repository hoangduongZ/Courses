# Basic Log Analysis - Practice Environment

Môi trường luyện tập xử lý log theo checklist thực chiến.

## Cấu trúc module

| Module | Kỹ năng | Thực hành |
|--------|---------|-----------|
| [01-log-locations](./01-log-locations/) | Vị trí file log trong `/var/log/` | Khám phá, phân loại log |
| [02-tail-realtime](./02-tail-realtime/) | `tail -f` theo dõi realtime | Script mô phỏng log stream |
| [03-grep-basics](./03-grep-basics/) | `grep` tìm lỗi cơ bản | Sample log + bài tập có đáp án |
| [04-severity-levels](./04-severity-levels/) | Log severity levels | Parse và phân loại theo level |

## Cách dùng

1. Đi vào từng folder theo thứ tự
2. Đọc `README.md` của module
3. Chạy script thực hành
4. Tự làm bài exercise trước khi xem đáp án

## Yêu cầu

- macOS hoặc Linux
- Bash 4+
- Không cần quyền root (môi trường mô phỏng hoàn toàn)
